# Лоб-ИИ-проект
## Жобаның мақсаты
Бұл жобаның мақсаты — Lobe AI платформасын қолдану арқылы суреттерді автоматты түрде танитын жасанды интеллект моделін құру және оның жұмыс істеу принципін зерттеу.
## Жобаның сипаттамасы
Жоба барысында суреттер жинақталып, олар бірнеше кластарға бөлінді. Lobe AI көмегімен модель оқытылып, кейін жаңа деректер арқылы тексерілді. Нәтижесінде модель суреттерді белгілі бір категориялар бойынша классификациялай алады.
